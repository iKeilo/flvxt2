# 隧道和节点筛选优化计划

## 问题描述
当隧道/节点筛选没有匹配的规则时显示空白，用户必须点击归零按钮才能再次筛选，体验不佳。

## 解决方案
1. 区分"无数据"和"无筛选结果"两种状态
2. 在筛选无结果时显示友好提示和归零按钮
3. 隧道/规则页面的隧道筛选下拉列表只显示有规则的隧道
4. 节点页面的隧道筛选下拉列表只显示有规则的隧道
