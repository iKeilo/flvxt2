    使用说明

    1. 仪表盘

    登录后先看到的是仪表盘，这里会展示系统概览：

    • 在线节点数量
    • 用户总数
    • 流量统计
    • 最近运行状态

    2. 节点管理

    节点是实际承载流量转发的机器。

    • 新增节点后会生成接入信息
    • 节点列表可以查看在线状态、版本、运行模式
    • 支持编辑、删除、重装、升级
    • 支持 blockOther 和实时连接数查看

    3. 用户管理

    • 可创建用户并分配流量配额
    • 支持用户组管理
    • 支持流量重置和到期管理

    4. 转发管理

    这是最核心的业务页面。

    • 支持 TCP / UDP
    • 支持 gost / nftables
    • 支持端口转发和隧道转发
    • 列表可查看实时连接数、实时速率、运行模式
    • 详情弹层可查看节点级实时明细

    5. 限制与策略

    • 可按用户或规则设置限速、连接数和流量限制
    • 支持流量统计与重置
    • 支持节点分组、节点标签和分组权限

    6. 系统配置

    这里可以设置：

    • 站点名称
    • Logo / Favicon
    • github_proxy
    • license 相关配置
    • 其他全局参数

    7. 个人设置

    • 修改密码
    • 查看个人信息
    • 切换部分显示配置
